# javascript-web-htmx-get-mustache-python-flask-api-surrealdb-simple

## Description
A demo of htmx using a python flask
api to return contents of table from
surrealdb.

## Tech stack
- htmx
    - get
    - ext
    - swap
    - target
- mustache
- python
    - flask
    - cors
- surrealdb

## Docker stack
- httpd:latest
- python:latest
- surrealdb/surrealdb:latest

## To run
`sudo ./install.sh -u`
- Available at http://localhost

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
